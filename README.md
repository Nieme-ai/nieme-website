# Nieme Website

The public website for [Nieme](https://nieme.ai).

Nieme is building a human-governed runtime for continuous execution:
persistent machine roles that perform work, produce evidence, submit
artifacts for review, and carry approved decisions forward.

## Repository contents

- Public website pages
- Website styling and interactions
- Brand glyph and favicon assets
- Vercel deployment configuration
